1 - обратить внимание, селект (Тип мотоцикла) первый options  скрывается с помощью js  и должен всегда находится на первом месте.
2 - в связаных списках нужно учитывать что 3-й  список орейнтируется по id , там в value должен быть id , так же в объекте должны быть
первыми элементами (название селекта - например Город  , вторым - сброс) это сделано для того, чтобы была кнопка сброса как по дизайну и
по нажатию подкидывался первый елемент, который по умолчанию скрыт. Так же учесть что ключ должен начинатся с цыфры 3 , потому что первый будет всегда
елемент по умолчанию, а второй кнопка сброса.
3 - в подборе модели и марке используются шаблоны которые в js , туда нужно передавать обхекты и все будет выводится астоматически,
для отображенение марки используется 2-ва объекста , так же и для модели. 1-й для популярных, второй для полного списка.
